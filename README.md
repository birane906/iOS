## Cadre
Le projet a été réalisé dans le cadre de la formation IG4 à Polytech Montpellier. L'objectif était de réaliser une application mobile IOS pour permettre aux visiteurs du Festival du Jeu de Montpellier de s'orienter.  
**Le groupe d'étudiant qui a réalisé l'application :**
- BA Biranne
- BOURRET Raphaël
- XIANG Jingjing

Parallèlement, le groupe a développé une application web pour permettre aux organisateurs de gérer tous leurs festivals.  
 
**Le lien du Git de l'application web :** [https://github.com/birane906/AWI](https://github.com/birane906/AWI)

## Installation locale

### Prérequis

- Un Mac
- XCode
- Une base de données postgres

### Installation

Ouvrir un terminal et rentrer les commandes suivantes :
```
git clone https://github.com/birane906/iOS.git IOS
```

Ouvrir le dossier `IOS` avec XCode et lancer l'application (petite flèche en haut à gauche)

Votre application devrait s'ouvrir dans un émulateur d'IPhone.  
Pour remplir votre base de données vous trouverez 2 fichiers `*.backup` à la racine du projet. Restaurez en premier le fichier `schema` et le fichier `data` ensuite.
